## Top 3 quick wins (ship today, ~5 min total)

1. **Bottom nav has phantom 4th column** ‚Äî `pages/MobileApp.tsx:954` has `grid-template-columns: repeat(4, 1fr)` but the nav now renders only 3 items. The 3 tabs appear left-biased with dead space on the right. **Fix:** change `4` ‚Üí `3`.

2. **CrearPersonaje toasts use voseo** ‚Äî `pages/CrearPersonaje.tsx:510` and `:676` still have `'Escrib√≠ un nombre'` and `'Escrib√≠ qu√© cambiar'`. Every new user hits these. **Fix:** `Escrib√≠` ‚Üí `Escribe` (2 strings).

3. **Recast resolution toggle is vestigial** ‚Äî Configure phase shows 480p/720p toggle but Kling 3 motion-control ignores it. Users select 480p expecting cheaper/faster, get identical output at same cost. **Fix:** hide the toggle until we wire an Express/Pro tier with Wan VACE.

## Suite-level recommendations

### Patterns to extract
1. **Custom topbars in Sesi√≥n + Reimaginar ‚Üí migrate to `AppTopBar`** (~50 lines saved, instant consistency)
2. **Video phase shell** ‚Äî Recast/Reels/VideoEdit duplicate pick‚Üíconfigure‚Üígenerating‚Üíresult. A `useVideoPhase` hook would eliminate ~300 lines.
3. **`creditCosts.ts` central file** ‚Äî `COST`, `COST_PER_SEC`, `COST_FULL_SHEET`, `PREMIUM_EXTRA` are magic numbers scattered across 10 files.

### Inconsistencies to reconcile
- **Back navigation target:** 4 apps use `'studio'`, Imagina uses `'home' as Page`, 3 newer apps abstract via `AppTopBar`. Pick one canonical.
- **Dark mode:** 3/10 apps. Next to wire: HeadshotPro + Reimaginar (highest traffic).
- **Prompt language:** Reels/VideoEdit templates in Spanish, HeadshotPro/Sesi√≥n prompts in English. Audit which engines are language-sensitive.

### Best-in-class app to model after
**HeadshotPro** ‚Äî uses all shared components, best progress labels, distinguishes content-policy errors from generic, restores credits properly. Other apps should look more like this one.
